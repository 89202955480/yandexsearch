Настройки подключения к API

API-ключ

Ключ для доступа к поиску. О том, как получить ключ см. раздел Доступ к API

SearchID

Идентификатор поиска. Чтобы узнать идентификатор, выберите поиск на странице Мои поиски
Идентификатор указан в адресной строке страницы поиска: https://site.yandex.ru/catalogs/<идентификатор_поиска>

Адрес YML-файла

Для формирования YML-файла рекомендуется использовать плагин Яндекс.Маркет
Адрес YML-файла необходимо указать в настройках на яндексе
